# 画面一覧

- メモ一覧
    - 既存のメモの一覧が表示されている
    - メモをクリックすることによって、そのメモのキャンバス画面に移動
    - 新規追加ボタンをクリックすることによって空のキャンバス画面に移動
- キャンバス画面
    - 矩形や、曲線ツールを用いて自由にメモを作成できる
    - キャンパス内に設置した矩形やオブジェクトは好きに色や枠線を設定できる
    - パスを用いて入力欄を設定できる
    - 「メモに画像を追加」ボタンをクリックすることによってキャンパス画面 (画像切り抜き)に移動。
    - オブジェクト同士のbool結合が可能
- キャンパス画面 (画像切り抜き) ※ 旧プロジェクトにも存在
    - 画像をパスで囲うことによって切り抜きが可能
    - 画像をパスで囲った上で、「メモ化」をクリックすることによって、切り抜き画像を作成し、キャンパス画面 (画像調整)に移動
- キャンパス画面 (画像調整) ※ 急プロジェクトにも存在
    - 画像の調整が可能
